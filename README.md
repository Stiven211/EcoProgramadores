# 🌱 EcoProgramadores STEAM MUTIS

Página web del grupo de estudiantes del Colegio Celestino Mutis que crean proyectos innovadores en ecología, programación, arte y matemáticas para un futuro sostenible.

## 📋 Descripción

Somos un grupo de estudiantes del Colegio Celestino Mutis, liderados por la profesora **Nathalie Rosales C.**, parte del programa STEAM Education. Creamos proyectos que combinan tecnología y ecología para un futuro más verde y sostenible.

## 🛠️ Tecnologías Usadas

- **HTML5** - Estructura de la página
- **CSS3** - Estilos y diseño responsivo
- **JavaScript** - Funcionalidad interactiva
- **EmailJS** - Formulario de contacto

## 📁 Estructura del Proyecto

```
EcoProgramadores/
├── index.html                    # Página principal
├── styles.css                   # Estilos CSS
├── script.js                    # Funcionalidad JavaScript
├── README.md                    # Este archivo
├── proyecto-animacion.html      # Página del proyecto animación
├── proyecto-juego.html          # Página del proyecto juego
├── proyecto-podcast.html       # Página del proyecto podcast
├── proyecto-prototipo.html      # Página del proyecto prototipo
├── proyecto-diseno.html         # Página del proyecto diseño
├── proyecto-presentacion.html   # Página del proyecto video
├── proyecto-educativo.html      # Página del proyecto educativo
├── logo/                       # Imágenes del logo
├── img-lider-integrantes/       # Fotos del equipo
└── Ecoprogramadores/            # Archivos de los proyectos
```
